## 功能技术
* 登录登出
* 功能菜单
* Dashboard
* 用户管理
* 注册
* 主题切换
* JWT
* Json转Dart Model类

## 安装
```bash
flutter packages get
flutter create .
```

## 运行
```bash
flutter run -d chrome
```
